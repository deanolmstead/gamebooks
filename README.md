# Gamebooks

A collection of interactive gamebooks — choose-your-own-adventure fiction playable in any browser. Each book is a self-contained HTML file with no build step required.

## Books

| Title | Setting | Status |
|-------|---------|--------|
| [Ashfall](./ashfall/) | San Joaquin Ashlands — post-apocalyptic dust-bowl survival | In Progress |

## Playing

Open any `index.html` directly in your browser, or visit the GitHub Pages link for each book.

## Structure

```
gamebooks/
└── ashfall/
    └── index.html   # Self-contained, playable in browser
```
